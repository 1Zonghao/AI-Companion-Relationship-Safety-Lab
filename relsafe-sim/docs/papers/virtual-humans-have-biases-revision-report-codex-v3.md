# 《虚拟人也有偏见》Codex v3 修改说明

## 1. 修改说明

- 以 `virtual-humans-have-biases-SH_T-codex-v2.docx` 为初版，保留题目、核心命题、冻结结果、负面发现、声明与参考文献。
- 将原来的“相关研究—理论—系统—三个实验—讨论—原则—局限”重组为九个主章节，使叙事从研究问题进入方法，再由三个实验支撑讨论与规范。
- 引言严格改为四段：样本问题、AI 陪伴场景、方法与三个发现、核心命题与贡献。
- 每个实验先给出用户指定的白话结论，再报告数值、分类和限制。
- 首次出现的关键英文术语给出中英文，此后优先使用中文；内部枚举集中在表格、括号或复现说明。
- 十条原则压缩为六条正文原则，并增加最低报告清单表。
- 十二条局限合并为外部效度、测量、工程复现和理论四类。
- 结论压缩为三段，没有新增统计显著性或因果主张。
- 参考文献 [1]–[8] 未增、未删、未改。

1. 旧 `glm-4-flash` 在三个 Policy 下均为 0.30，因未体现 system prompt 差异而排除；不作安全推断。
2. `glm-4.7-flash` 因调用过慢排除；不作安全推断。

5. 当前未观察到 Rank Dependence 或 Conclusion Dependence，但不能外推为其他模型不会反转。

7. Reviewer A/B 总体 NA 比例分别为 43.28% 和 46.12%；C 组均为 65.91%，D 组均为 95.45%。

10. A1 规则零召回：0 TP、2 FP、5 FN。
11. A3 规则零召回：8 FN。

14. C1 Judge/Ensemble 的 F1 虽为 1.000，但只来自一个正例。
15. C5 Ensemble 漏掉全部 8 个正例，F1 为 0。
16. A1、A3 rule、A5、C3 与 C5 仍为低置信标签。
17. 44 条 pilot items 不能被解释为稳定性能估计。
18. MiniMax/Kimi 的差异不能外推为真实人群或心理依赖。
19. 当前不能证明 AI 陪伴导致心理依赖。
